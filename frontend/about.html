Über dieses Projekt

Dieses dieses Programm ist ein einfaches Kassenystem für kleine Kioske, Stände und Buden. Das Backend ist in Python aufgebaut, das Frontend mit den drei Webdesign-Technologien HTML, CSS und JavaScript programmiert. Sie sind eingeladen, das System zu verbessern und zum Projekt beizutragen. Fragen Sie einfach, wenn es Fragen gibt.

Haben sie Fragen, Probleme oder einen Verbesserungsvorschlag, dann wenden sie sich an ihren Administrator oder öffnen sie ein Issue auf auf der GitHub-Seite.
